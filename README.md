# RTSP Stream Analyzer v1.0

RTSP Stream Analyzer - это Bash-скрипт для глубокого анализа и мониторинга качества потоков с IP-камер. В отличие от обычных чекеров, он измеряет физический битрейт, анализирует метаданные и оценивает качество изображения.

Этот инструмент идеально подходит для системных администраторов и инженеров видеонаблюдения. 

Для работы скрипта на Arch Linux (ARM64) или любом другом дистрибутиве необходимы:

## Установка зависимостей
> [!IMPORTANT]
> sudo pacman -S ffmpeg bc gnu-netcat iputils

## Запуск и использование
Скрипт поддерживает два режима запуска: с передачей параметров и с использованием настроек по умолчанию из тела скрипта.

### Способ 1: Прямая передача аргументов (Рекомендуется)

> [!TIP]
>./rtsp_analyzer.sh "rtsp://user:pass@192.168.1.100:554/live" 5
> 
> Аргумент 1: Полная RTSP ссылка (обязательно заключайте в кавычки, если в пароле есть спецсимволы).
> 
> Аргумент 2: Время замера битрейта в секундах (например, 5).

### Способ 2: Использование значений по умолчанию
> [!TIP]
> Если запустить скрипт без параметров, он возьмет настройки из переменных URL и DURATION, указанных внутри файла:
> 
> ./rtsp_analyzer.sh

Совет: Если вы планируете мониторить одну и ту же камеру постоянно, просто отредактируйте блок - Настройки - в начале скрипта.
